# Control Statements

## If

```buildoutcfg
if <Conditions>:
    <Statements>
```

## If..Else
```buildoutcfg
if <Conditions>:
    <Statements>
else:
    <Statements>
```

## If..Else If...Else
```buildoutcfg
if <Conditions>:
    <Statements>
elif <Conditions>:
    <Statements>
.
.
.
else:
    <Statements>
```
